Aprendendo Negrito e Itálico

Para se colocar uma palavra, frase ou até mesmo um texto em negrito devemos usar a tag >strong< que dará sentido no html e que vai deixar em negrito.

Para se colocar uma palavra, frase ou até mesmo um texto em Itálico devemos usar a tag >em< que dará sentido no html e que vai deixar em Itálico.

As tags >strong< e >em< servem para dar sentido em HTML, ou seja, servem para dar semântica, oque é fundamental dentro de html5
