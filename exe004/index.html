Parágrafos/Quebras de linha!

Vamos quebrar linhas dentro de um único parágrafo, não basta dar enter ao digitar o código para a linha ser quebrada, temos que inserir a tag br.

Atenção! Precionei enter, entre as linhas da tag <p> ,porém, não significa que vai realmente quebrar as linhas dentro do site sendo mostrado no navegador apenas como uma única linha. Para quebrar a linha e ser mostrado no navegador a linha quebrada, será usado uma tag específica que irei mostrar no próximo exemplo.
