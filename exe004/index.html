Parágrafos/Quebras de linha!

Olá, tudo bem? Irei mostrar como fazer a quebra de linhas dentro de um único parágrafo, basta dar enter ao digitar o código e a linha será a quebrada, mas para isto funcionar tem que estar entre o par de tags br.

ATENÇÃO! Eu precionei enter entre as linhas da tag <p> ,porém, não significa que vai realmente quebrar as linhas dentro do site sendo mostrado no navegador apenas como uma única linha. Para quebrar a linha e ser mostrado no navegador a linha quebrada, será usado uma tag específica que irei mostrar no próximo exemplo.
